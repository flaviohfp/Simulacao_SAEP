Nova tarefa

O status inicial da tarefa sera cadastrado como "a fazer".

O que o sistema entrega

  • Cadastro de usuarios com nome e e-mail.
  • Cadastro de tarefas com descricao, setor, prioridade e responsavel.
  • Status inicial definido como "a fazer".
  • Edicao, exclusao e alteracao de status na tela principal.
  • Dados gravados em banco PostgreSQL com script SQL na pasta do projeto.
  • Arquivos de DER e caso de uso salvos na pasta public/assets/img.
