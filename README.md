# Simulacao_SAEP
simulacao do SAEP

## Banco de dados

O projeto usa PostgreSQL.

Defina estas variaveis antes de iniciar:

- `PGHOST`
- `PGPORT`
- `PGUSER`
- `PGPASSWORD`
- `PGDATABASE`

Ou use `DATABASE_URL`.

Ao iniciar o servidor, o arquivo [schema.sql](C:\Users\flavi\OneDrive\Documentos\GitHub\Simulacao_SAEP\database\schema.sql) cria as tabelas e insere os dados iniciais.
